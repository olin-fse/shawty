# Week 1 - Unit Testing

## Golang

To run Go tests:

```
cd ~/go/src/github.com/olin-fse/shawty
go get ./...
ginkgo
```

The Go test file is `main_test.go`

## Javascript

To run Jest tests:

```
cd ~/go/src/github.com/olin-fse/frontend
npm install
npm test
```

The Jest test file is `test/api.spec.js`
